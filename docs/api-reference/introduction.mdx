## Key Types

| Type | Description |
|---|---|
| `AgentConfig` | Builder for configuring agent behavior, tools, hooks, and LLM settings |
| `AgentHandle` | Handle for communicating with a spawned agent (input, subscribe, interrupt) |
| `AgentRuntime` | Manages agent lifecycles and spawning |
| `AgentSession` | Persistent session with conversation history |
| `StandardAgent` | Default agent implementation with the standard tool-use loop |
| `OutputChunk` | Enum of all events emitted by agents (text, tools, permissions, etc.) |
| `InputMessage` | Enum of all messages you can send to agents |
| `ToolRegistry` | Container for registering tools |
| `Tool` | Trait for implementing custom tools |
| `ToolResult` | Return type from tool execution |
| `PermissionRule` | Allow/deny rules for the permission system |
| `HookRegistry` | Container for registering lifecycle hooks |
